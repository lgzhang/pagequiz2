平抛物体运动

2．方法梳理

方法一：“化曲为直”

即把曲线运动分解为两个直线运动．研究平抛运动时，将其分解为水平方向直线运动和竖直方向的直线运动，水平方向是匀速直线运动，竖直方向上是自由落体运动．

方法二：平抛物体运动的处理方法及规律

以物体抛出点为坐标原点，以初速度υ0方向为${ox}$轴正方向，以竖直向下为${oy}$轴正方向，建立平面直角坐标系，如图所示．

（1）速度规律：${{\upsilon _x} = {\upsilon _0}}$，\({\upsilon _y} = gt\)，合速度大小\(\upsilon = \sqrt {\upsilon _x^2 + \upsilon _y^2} = \sqrt {\upsilon _0^2 + {{(gt)}^2}} \)，任意时刻的速度υ与υ0的夹角为\(\theta \)，则${\tan \theta = \dfrac{{{\upsilon _y}}}{{{\upsilon _x}}} = \dfrac{{gt}}{{{\upsilon _0}}}}$．

（2）位移规律：${x = {\upsilon _0}t}$，\(y = \dfrac{1}{2}g{t^2}\)，合位移大小${s = \sqrt {{x^2} + {y^2}} = \sqrt {{{({\upsilon _0}t)}^2} + {{(\dfrac{1}{2}gt)}^2}} }$，方向${\tan \varphi = \dfrac{y}{x}}$．

（3）平抛运动的飞行时间：${t = \sqrt {\dfrac{{2y}}{g}} }$（只与下落高度y有关，与其他因素无关）．

（4）平抛运动的水平距离：${s = {\upsilon _0}\sqrt {\dfrac{{2y}}{g}} }$（决定于初速度和高度）．

（5）平抛运动在任意时刻的瞬时速度υt的反向延长线一定通过此时水平总位移x的中点，即${x' = \dfrac{1}{2}{x_t}}$，利用这一推论，可以极为简捷地确定平抛运动任意时刻的速度方向．

（6）平抛运动任意时刻的速度偏向角的正切函数值tan${\theta }$，等于位移偏向角（总位移与υ0的夹角）的正切值tanφ的2倍，即tan${\theta }$＝2tanφ．证明：${\tan \theta = \dfrac{{{\upsilon _y}}}{{{\upsilon _0}}} = \dfrac{{gt}}{{{\upsilon _0}}} = \dfrac{{\dfrac{1}{2}g{t^2}}}{{\dfrac{1}{2}{\upsilon _0}t}} = 2\dfrac{{{y_t}}}{{{x_t}}}}$．
